Creates Fig 3 in

Espen Hagen, David Dahmen, Maria L. Stavrinou, Henrik Lindén, Tom Tetzlaff,
Sacha J van Albada, Sonja Grün, Markus Diesmann, Gaute T. Einevoll.
"Hybrid scheme for modeling local field potentials from point-neuron networks".
arXiv:1511.01681v2 [q-bio.NC]. http://arxiv.org/abs/1511.01681v2.
Contact: e.hagen@fz-juelich.de.


Usage (after installing all prerequisites, see above README.txt):
    python Fig3.py
    
The script will generate the pdf 'Fig3.pdf'
